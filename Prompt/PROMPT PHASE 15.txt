Je veux connecter l’app mobile Expo à l’authentification JWT du backend.

Fais les actions suivantes :

1. Crée un fichier /mobile/lib/auth.ts :
   - login(email, password)
   - register(email, password)
   - saveToken(token) → AsyncStorage
   - getToken()
   - logout()

2. Mets à jour /mobile/lib/api.ts :
   - Ajoute automatiquement Authorization: Bearer <token>

3. Mets à jour les écrans :
   /mobile/app/login.tsx
   /mobile/app/index.tsx
   /mobile/app/progress.tsx
   /mobile/app/lesson/[id].tsx
   /mobile/app/profile.tsx

4. Ajoute un hook useAuth() :
   - vérifier si l’utilisateur est connecté
   - récupérer userId depuis le token
   - rediriger si non connecté

5. Ajoute un bouton Logout dans l’écran Profil.

Le code doit être propre, typé, modulaire.
