Je veux implémenter la logique complète des leçons dans l’app mobile Expo.

Objectif :
- gérer plusieurs types d’exercices
- afficher un exercice à la fois
- gérer les réponses
- envoyer les réponses au backend
- afficher score, erreurs, XP

Fais les actions suivantes :

1. Dans /mobile/components/exercises/, crée :
   - MultipleChoiceExercise.tsx
   - TranslationExercise.tsx
   - WordOrderExercise.tsx
   - ListeningExercise.tsx
   - FillInTheBlankExercise.tsx

2. Dans /mobile/lib/exerciseTypes.ts :
   export type ExerciseType = "mcq" | "translate" | "order" | "listen" | "fill";

3. Dans /mobile/app/lesson/[id].tsx :
   - charge la leçon via API
   - charge les exercices
   - crée un state :
       currentIndex
       answers[]
       isFinished
       result
   - affiche le bon composant selon exercise.type
   - gère la progression
   - soumet les réponses à la fin

4. Crée un écran LessonResult.tsx :
   - affiche score
   - erreurs
   - XP gagné
   - bouton retour

5. Ajoute un composant ExerciseWrapper.tsx pour :
   - barre de progression
   - numéro d’exercice
   - wrapper visuel

6. Le code doit être propre, typé, modulaire, et compatible avec les futures variantes.
