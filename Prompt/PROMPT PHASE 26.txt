PHASE 23 — Analytics & Monitoring (Backend + Web + Mobile)

Je veux ajouter un système complet d’analytics et de monitoring.

==========================
1. BACKEND
==========================

1.1 Ajouter un module Analytics :
    - loguer :
        - ouverture de leçon
        - soumission de leçon
        - progression
        - révision
        - gamification
    - stocker dans une table AnalyticsEvent :
        id, userId, type, payload JSON, timestamp

1.2 Ajouter un module Monitoring :
    - temps de réponse moyen
    - erreurs par minute
    - requêtes par endpoint
    - uptime

1.3 Ajouter un endpoint /metrics compatible Prometheus

==========================
2. WEB
==========================

2.1 Ajouter un tracker frontend :
    - page views
    - clics importants
    - début de leçon
    - fin de leçon
    - erreurs

2.2 Envoyer les events au backend via /analytics

2.3 Ajouter un hook useAnalytics()

==========================
3. MOBILE
==========================

3.1 Ajouter un tracker mobile :
    - ouverture d’écran
    - interactions
    - début/fin de leçon
    - erreurs

3.2 Envoyer les events au backend

3.3 Ajouter un hook useAnalytics()
