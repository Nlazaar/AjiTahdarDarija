PHASE 21 — Optimisation des performances (Backend + Web + Mobile)

Je veux optimiser les performances de toute l’application (backend, web, mobile).

==========================
1. BACKEND — OPTIMISATION
==========================

1.1 Activer la compression gzip
1.2 Activer le cache HTTP sur les endpoints statiques
1.3 Ajouter un cache Redis pour :
    - modules
    - lessons
    - exercises
    - gamification
    - review items
1.4 Ajouter un cache interne (TTL) pour les requêtes fréquentes
1.5 Optimiser Prisma :
    - select / include minimal
    - index sur les champs utilisés
    - pagination
1.6 Ajouter un middleware de performance :
    - mesurer le temps de chaque requête
    - loguer les requêtes lentes (> 300ms)

==========================
2. WEB — OPTIMISATION
==========================

2.1 Activer le rendu statique (SSG) pour :
    - home
    - login
    - register

2.2 Activer ISR (Incremental Static Regeneration) pour :
    - modules
    - lessons

2.3 Optimiser les images :
    - next/image
    - formats WebP

2.4 Ajouter un cache local (localStorage) pour :
    - modules
    - lessons
    - gamification

2.5 Ajouter un hook usePrefetch() :
    - précharger la prochaine leçon
    - précharger les exercices

2.6 Ajouter un système de lazy loading pour les composants lourds

==========================
3. MOBILE — OPTIMISATION
==========================

3.1 Activer le caching via AsyncStorage :
    - modules
    - lessons
    - gamification

3.2 Ajouter un système de prefetch :
    - précharger les exercices de la prochaine leçon

3.3 Optimiser les animations :
    - react-native-reanimated
    - useNativeDriver

3.4 Optimiser les listes :
    - FlatList optimisée
    - keyExtractor
    - getItemLayout

3.5 Ajouter un splash screen optimisé
