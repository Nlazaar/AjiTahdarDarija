PHASE 26 — Monétisation (Premium, pubs, abonnements)
Je veux ajouter un système complet de monétisation dans mon application.

==========================
1. BACKEND
==========================

1.1 Créer un module Billing :
    - endpoints :
        POST /billing/create-session
        POST /billing/webhook
        GET /billing/status
    - intégrer Stripe (abonnements)
    - plans :
        - Free
        - Premium Mensuel
        - Premium Annuel

1.2 Ajouter un champ dans User :
    - subscriptionStatus
    - subscriptionExpiresAt

1.3 Ajouter un guard PremiumGuard :
    - bloque l’accès aux fonctionnalités premium

==========================
2. WEB
==========================

2.1 Ajouter une page /premium :
    - comparaison Free vs Premium
    - bouton “Passer Premium”

2.2 Ajouter un composant Paywall :
    - affiché quand l’utilisateur n’est pas premium

2.3 Intégrer Stripe Checkout :
    - redirection vers la session de paiement

==========================
3. MOBILE
==========================

3.1 Ajouter un écran Premium :
    - comparaison Free vs Premium
    - bouton “Passer Premium”

3.2 Intégrer Expo In-App Purchases :
    - abonnement mensuel
    - abonnement annuel

3.3 Ajouter un Paywall mobile
